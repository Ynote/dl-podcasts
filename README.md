# Experiment Express

> Experiment from 2014

This is a small project that intends to test Webpack 1 and Jade in an Express application.

It is a app that display mp3 links from a list of podcasts

## Usage 

- Edit the `conf/podcasts.json`
- Launch the server
```
npm start
```
